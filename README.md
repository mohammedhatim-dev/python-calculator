# Python Calculator

A simple command-line calculator built using Python to practice basic programming concepts.

## Features
Basic Math: Supports Addition, Subtraction, Multiplication, and Division.

Basic Input Validation: Uses try/except to handle basic input errors (like entering text instead of numbers).

Zero Division Check: Prevents errors when dividing by zero.


## Technologies
Python 3 (Core Language)

Basic Control Flow (if/else statements)

Basic Error Handling (try/except blocks)
